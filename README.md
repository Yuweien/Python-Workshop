# Python Workshop

Grad-led Python workshop for absolute beginners.

This two-session workshop introduces participants to reading, understanding, and modifying Python code for research workflows. The focus is on practical data analysis using real-world CSV files (e.g., D2L exports), with an emphasis on reproducible research practices.
